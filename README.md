# NFT Real Estate Marketplace

A decentralized real estate marketplace where properties are represented as NFTs, allowing users to mint, list, buy, and transfer property assets on the Ethereum blockchain.

The platform integrates smart contracts with a modern web interface, enabling secure and transparent real estate transactions.

## Features
- Mint NFT-based real estate properties
- List and trade property NFTs on marketplace
- Connect wallet using MetaMask
- Buy, sell, and transfer NFTs via smart contracts
- Store NFT metadata on IPFS
- View owned assets in personal dashboard
- AI assistant for property suggestions (basic)

## Tech Stack
- JavaScript (Node.js)
- MongoDB
- Next.js, React
- RESTful API
- Solidity, Ethereum (Blockchain)
- Hardhat, MetaMask, IPFS
